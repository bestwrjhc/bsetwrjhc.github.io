●2.检查源节点/接入点是否设置正确<br />1）AUTO根据账号能力顶格播发<br />推荐使用，根据账号类型，自动播发信号。<br />2）RTCM23_G 不能使用。<br />3）RTCM30_GG播发双星信号<br />仅用于早些年不支持RTCM32老仪器使用，<br />如果账号是四星13频、五星16频的，选择该接入点只有双星的信号，稍微有点遮挡就浮动。<br />4）RTCM32_GGB播放三星信号<br />如果账号是四星13频、五星16频的，选择该接入点只有三星的信号，不能充分发挥账号多星优势。

### 212 [千寻GNSS]千寻GNSS 设置固定解方法


使用主机自带账号（千寻知寸）

●1.手薄插入SIM卡或连接手机wifi热点。

●2.连接仪器<br />点击【仪器】->【通讯设置】<br />蓝牙搜索主机SN，搜到以后点击链接。

●3.查看主机自带账号是否在有效期内<br />进入【千寻服务注册及查询】，在账号有效期内会显示到期时间，例如2022年12月31日到期。<br />（只有显示到期时间后，才能设置固定解）

●4.移动站数据链及模式设置<br />进入【移动站模式】，<br />数据链：选择手薄网络<br />连接模式：选择千寻知寸<br />坐标系：CGCS2000<br />点击应用。

●5.在空旷无遮挡环境稍等片刻，即可固定。

●6.注意：<br />1.到期时间显示“已到期”，需续费或输入NTRIP账号才能获取固定解。 <br />2.如果既不显示“已到期”也不显示“到期时间”，说明手薄没有网络，请连接wifi以后再尝试。<br /> <br />@220630更新<br />关键词：千寻GNSS设置固定解

### 213 [千寻GNSS]千寻GNSS 两个控制点计算转换参数：


千寻RTK测量的坐标为2000坐标，工程坐标一般为独立坐标，需要找两个控制点，把坐标转为当地坐标。<br />●1.新建项目、设置中央子午线。

●2.选取两个控制点。<br />尽量选择距离远、能把测区夹在中间的点位。<br />例如：A1、A2.

●3.测量控制点坐标。<br />1）测量A1。<br />进入【点测量】，对中杆升到1.8米，立在A1点位上，气泡保持居中不动。<br />点击“测量图标”，测量A1坐标，输入点名A1，杆高1.8米，点击确定保存。<br />2）测量A2。<br />立在A2点位上，气泡保持居中不动。<br />点击“测量图标”，测量A2坐标，输入点名A2，杆高1.8米，点击确定保存。

●4.计算转换参数<br />进入【工具】->【计算转换参数】<br />1）增加A1点。<br />已知点坐标（控制点）：输入A1已知坐标<br />当前WGS84坐标（测量点）：点击“已知”选择刚刚采集的A1坐标<br />点击确定保存<br />2）增加A2点。<br />已知点坐标（控制点）：输入A2已知坐标<br />当前WGS84坐标（测量点）：点击“已知”选择刚刚采集的A2坐标<br />点击确定保存

3）点击“计算”<br />检查 比例尺是否接近1.000或0.999，如果是点击应用。

●5.测量控制点，检查误差是否在允许范围内。

注意：<br />如果比例不是1.000或0.999，或检查控制点误差很大，按以下尝试<br />1.检查控制点位置、输入坐标是否正确。<br />2.新建项目选择本地坐标参数、修改中央子午线，再重新计算参数。<br /> <br />@220630更新<br />关键词：千寻GNSS两个点计算参数 千寻GNSS计算转换参数 千寻GNSS校点

### 221 [千寻GNSS]千寻星基使用说明
（仅SR3pro、SE星地一体版支持星基）


●1.激活星基服务<br />激活时需要用到网络，手薄插入SIM卡或连接手机热点<br />进入【千寻服务注册及查询】，往下看，找到“千寻星基服务”选项，<br />点击“激活服务”<br />稍等片刻，就会显示星基到期时间。

●2.检查星基服务是否已经在服务中<br />进入【千寻星基】图标，查看当前星基服务到期时间<br />服务状态：已在服务中<br />LSSR密钥失效时间：2025/3/31 18:00<br />只有服务状态显示已在服务中才可以使用。

### 9103 手薄如何输入带号？
220722更新

1）带号源自于高斯投影，<br />比如中央子午线99对应带号为33，114对应38，120对应40.

2）带号有什么用？<br />根据高斯投影定义，平面坐标北坐标7位，东坐标6位，例如：2731234.123,500123.123<br />如果单给一个坐标，我们并不知道在哪里？<br />如果在500123.123前面加上带号33，我们就知道对应的中央子午线为114，就可以找到在地球上找到这个点准确位置。

3）日常测量中不用带号可以吗？<br />带号在日常测量中没有什么意义，加上带号测量，东坐标太长，输入繁琐，建议测量时，统一把带号省去。

4）手薄如何输入带号？<br />在坐标系统里，投影参数里，动向加长数（50 0000）前，统一输入带号。
